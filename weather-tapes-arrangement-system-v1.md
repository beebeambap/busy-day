# The Weather Tapes — 테마 편곡 시스템 설계 v1.0

> 작성일: 2026-05-14  
> 상태: 1차 설계 초안 (RAIN 완료 / 나머지 날씨 조건 미정)

---

## 개요

The Weather Tapes의 주 콘텐츠는 날씨 데이터 기반 자동 생성 ambient 트랙이다.  
테마 편곡 시스템은 이 원곡들을 **날씨 조건별 감성 렌즈로 재해석**하여, 별도의 테마 앨범으로 구성하기 위한 알고리즘 레이어다.

---

## 핵심 원칙

### 1. 원곡 골격 유지
편곡은 원곡의 **조성(Key), 멜로디 음정 시퀀스, 마디 구조**를 고정한다.  
"익숙하면서도 새로운" 느낌은 이 골격을 유지한 채 리듬 문법과 질감만 교체함으로써 발생한다.

### 2. 감성 매핑 레이어
물리적 날씨 데이터를 직접 번역하지 않는다.  
**"이 날씨에 사람들은 어떤 음악을 듣고 싶어하는가"** 를 기준으로 장르 감성을 매핑한다.  
이것이 The Weather Tapes 편곡 시스템의 고유 설계 원칙이다.

### 3. 알고리즘 정체성 유지
편곡은 수동 작업이 아닌, **사전 설계된 규칙 세트의 실행**으로 정의된다.  
각 날씨 조건마다 고정된 편곡 파라미터가 존재하며, 이를 JSON 악보에 적용하는 방식으로 동작한다.

---

## 편곡 시스템 구조

```
원본 JSON 악보 (날씨 자동 생성)
        ↓
날씨 조건 파라미터 입력
        ↓
인간 감성 매핑 레이어 (The Weather Tapes 고유 설계)
        ↓
장르 프리셋 적용 (악기 구성 + 화성 언어 + 리듬 패턴)
        ↓
편곡된 JSON 악보 출력
        ↓
기존 렌더링 시스템으로 최종 트랙 생성
```

| 구분 | 항목 |
|------|------|

| | 악기 음색 |

## 날씨별 편곡 설계

---

### 🌧 RAIN

**핵심 이미지:**  
> "원곡을 비 내리는 카페 안에서 누군가 피아노로 다시 치는 느낌"

**장르 레퍼런스:** Acoustic Jazz + Indie Folk  
(Keith Jarrett의 고요함 + Novo Amor의 질감)

**편곡 파라미터:**

| 요소 | 원곡 기본값 | RAIN 편곡값 |
|------|------------|------------|
| 오른손 아티큘레이션 | 신스 레가토 | 피아노 레가토, 노트 끝 미세하게 흘리기 |
| 왼손 패턴 | 패드 기반 지속음 | 낮은 루트음 → 상위 코드톤 아르페지오 (재즈 워킹 감각) |
| 리듬 그루브 | Straight | 미세 스윙 적용, 비트를 약간 뒤로 |
| 벨로시티 분포 | 균일 | 오른손 pp~mp / 왼손은 더 낮게 (멜로디가 수면 위로 뜨는 느낌) |
| 화음 보이싱 | 기본 트라이어드 | 오픈 보이싱 + 9th 추가 (공기감) |
| 악기 음색 | 신스 패드 | 어쿠스틱 피아노 또는 Rhodes |
| 템포 | 원본 | 원본 대비 -10~15% |

**추가 레이어 (옵션):**
- 기타 하모닉스 (멜로디 보조, 낮은 볼륨)
- 빗소리 텍스처는 **원곡에 이미 포함**되어 있으므로 별도 추가 없음
